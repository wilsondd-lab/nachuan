# 纳川.skill

> To be anyone, to be better you.

**Distill Everything.skill**

一切可得的材料，先扫描，后提问，四路管线产出。

```bash
npx skills add wilsondd-lab/nachuan
```

---

## 做什么

你在对话里扔进一份 PDF、一通会议录音转写、一份竞品分析 PPT，或者零散的十几条笔记 — 它不急着问你要什么，而是先看清材料里有什么，再把可走的路线摆到你面前。

**四条路：**

| 路线 | 做什么 |
|---|---|
| A · 精华提炼 | 抓核心论点、关键判断、可复用的模式 |
| B · 结构拆解 | 整理叙事框架、表达结构、内容骨架 |
| C · 知识提取 | 输出可索引的知识笔记，带标签和双向链接 |
| D · 内容生成 | 基于提取结果生成文案、大纲、脚本 |

可单选，可并行，可串联。

---

## 怎么用

```text
帮我看看这个文档里有什么
把这份 PPT 拆一下
提取里面能复用的知识点
```

把材料拖进对话窗口，它会先交一份结构概要，再给你 2-4 个处理选项。你选一个，它往下走。

---

## 输出

默认生成 Markdown 知识笔记，放在 `纳川/` 目录下。每条笔记带 YAML 标签和 `[[双向链接]]`，适合直接接入任何 Markdown 笔记系统。
