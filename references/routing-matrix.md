## 路径组合模式速查

| 模式 | 符号 | 何时用 |
|------|------|--------|
| 单路径 | A / B / C / D | 用户目标单一明确 |
| 双并行 | B+C / A+C | 用户要"都要"，两条线不依赖 |
| 三并行 | A+B+C | PPT 有讲稿+演讲者，三线全开 |
| 串联 | C→D / C→A | 后一步依赖前一步的 vault 输出 |
